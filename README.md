# Cards
Учебный проект. Игра, в которой нужно выбрать 2 одинаковые карточки.

В ходе проекта изучены:
- UIView, CALayer
- CoreAnimation и CoreGraphics
- Кривые Безье
- Создание анимаций
- Верстка кодом
- MVC
- Шаблон "Фабрика"
